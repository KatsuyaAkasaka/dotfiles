# How to setup iterm

## Step.1

open `iTerm2` app and open preference. (shortcut: `cmd`+`,`)

## Step.2

open `Preferences` tab in `General` page and click `Browse` button.
then, choose `com.googlecode.iterm2.plist` file in `~/dotfiles/app/iterm`
