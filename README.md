# dotfiles

##
run bellow to install develop tool.
```
git
```

## Step.1
copy setup.sh to your local machine on home directory and run this
```
sh setup.sh
```


## Step.2
check successfuly cloned this repo and do this
```
cd dotfiles
```


## Step.3
just do this
```
sh installer.sh
```


## Step.4
your machine setup is totally finished!
